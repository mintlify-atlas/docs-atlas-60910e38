# Welcome to Better Home

Better Home is a minimal, delightful new-tab replacement for Chromium-based browsers. It transforms your new tab into a productivity hub with everything you need and nothing you don't.

<CardGroup cols={2}>
  <Card title="Tasks" icon="checkbox">
    A lightweight to-do list that remembers your items across sessions. Add, check-off, and delete tasks in a snap.
  </Card>
  <Card title="Quick Links" icon="link">
    One-click bookmarks with auto-fetched favicons. Save your most-visited sites and jump to them instantly.
  </Card>
  <Card title="Mood Calendar" icon="calendar">
    Track how you feel every day of 2026 with a beautiful, color-coded calendar. Supports quadrimester and full-year views with smooth Motion animations.
  </Card>
  <Card title="Customizable" icon="settings">
    Enable or disable any widget via the extension popup - make it your home. Includes backup & restore functionality.
  </Card>
</CardGroup>

## Key Features

<Info>
Better Home is built with modern web technologies: React 19, TypeScript 5.9, Vite 7, and Tailwind CSS 4.
</Info>

### Fully Responsive
Works seamlessly from ultrawide monitors to small laptops. The layout adapts to your screen size automatically.

### Data Persistence
All your tasks, links, calendar entries, and settings are stored locally in your browser using localStorage. Your data never leaves your machine.

### Backup & Restore
Download your data as JSON to backup or transfer to another browser. Upload to restore your tasks, links, calendar entries, and settings.

### Smooth Animations
Built with Motion (v12) for delightful, performant animations throughout the interface.

### Mood Tracking
The mood calendar supports:
- Color-coded mood entries
- Quadrimester and full-year views
- Optional day-number toggle
- Inline notes for each day
- Smooth view transitions

## What You'll Need

To install and use Better Home, you'll need:

<Steps>
  <Step title="A Chromium-based browser">
    Better Home works with Chrome, Edge, Brave, and other Chromium-based browsers.
  </Step>
  <Step title="Bun package manager">
    Required to build the extension from source. See the [installation guide](/installation#prerequisites) for setup instructions.
  </Step>
  <Step title="Developer mode enabled">
    You'll need to enable Developer mode in your browser's extension settings to load the unpacked extension.
  </Step>
</Steps>
